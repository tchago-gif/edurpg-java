# EduRPG Java

Sistema educacional gamificado desenvolvido em Java para aplicar conceitos matemáticos em um contexto de aprendizagem baseado em RPG.

## Funcionalidades

- Gerenciamento de alunos
- Cadastro, edição, remoção e listagem
- Registro de desempenho em missões
- Matriz de desempenho
- Cálculo de XP, nível e aproveitamento
- Classificação dos alunos
- Operações com conjuntos
- Cálculo de combinações de equipes
- Simulação de recompensas com probabilidade

## Conceitos aplicados

- Conjuntos
- Funções e métodos
- Lógica condicional
- Análise combinatória
- Probabilidade
- Vetores
- Matrizes
- Laços de repetição

Compile o arquivo:

```bash
javac EduRPG.java
